##Para que la app se sienta profesional y "limpia", usaremos una estética moderna centrada en la legibilidad.

""Paleta de Colores
Primario (Acción): #2D5BFF (Azul confianza/finanzas).

Éxito (Ingresos): #00C853 (Verde esmeralda).

Peligro (Gastos/Alertas): #FF3D00 (Naranja rojizo).

Fondo: #F8F9FA (Gris muy claro).

Texto: #212121 (Casi negro para alto contraste).

##Tipografía
Principal: Inter o Montserrat (Sanson-serif modernas).

Cuerpo: 16px para legibilidad.

Títulos: Semibold para jerarquía clara.

#Componentes Clave
Cards: Para mostrar el resumen de saldos.

Modales: Para el ingreso rápido de gastos.

Inputs Dinámicos: Con validación de moneda (pesos colombianos).
