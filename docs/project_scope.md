""Visión General
Aplicación web para la gestión financiera compartida, permitiendo una distribución equitativa de gastos basada en el nivel de ingresos de cada integrante del grupo familiar.

Funcionalidades Principales
##Gestión de Usuarios y Grupos:

Autenticación mediante Firebase (Email/Password y Google).

Creación de "Grupos Familiares".

Sistema de invitaciones para unir a la pareja al grupo.

Roles: Administrador (gestiona categorías y miembros) y Miembro.

##Cálculo de Aportes Proporcionales:

Registro de salarios individuales.

Cálculo automático del porcentaje de aporte (Ej: Si A gana 70 y B gana 30, los gastos se dividen 70/30).

##Gestión de Movimientos:

Registro de ingresos y gastos con fecha, categoría y descripción.

Repartición automática de cada gasto ingresado según el porcentaje calculado.

La interfaz debe ser "Mobile First" (pensada para celulares).

##Visualización de Datos:

Dashboard con gráficos dinámicos (Gasto por categoría, tendencia mensual).

Resumen de "quién le debe a quién" o balance del fondo común.

##Fuera de Alcance (Out of Scope)
Conexión directa con API bancarias (lectura de extractos).

Pagos reales entre usuarios dentro de la app.
