React.js	Biblioteca principal para construir la interfaz de usuario (UI) de forma modular.
Firebase Auth	Manejo de registro, inicio de sesión y seguridad de usuarios.
Cloud Firestore	Base de datos NoSQL en tiempo real para guardar ingresos, gastos y grupos.
CSS (Tailwind o Modules)	Estilizado rápido y responsive para que la app se vea bien en celulares.
Chart.js / Recharts	Librería para generar los gráficos dinámicos mensuales.
Git / GitHub	Control de versiones y despliegue (puedes usar GitHub Pages o Firebase Hosting).
